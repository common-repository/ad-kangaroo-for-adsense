Ad Kangaroo for Adsense - Probably the quickest way to insert Google AdSense & other ads into your website. Google AdSense and support for Google AMP.

== Description ==

Ad Kangaroo plugin is a superb solution to earn money by placing AdSense ads to WordPress website pages, posts, custom posts, categories, tags, and widgets.

Link your website to Google AdSense account and manage available ads easily right from your admin dashboard!

#### Ad Kangaroo for AdSense
Ad Kangaroo for AdSense is a free Google AdSense plugin that can be used to easily insert your Google AdSense ads into your posts, pages, categories and even tag pages.

Ad Kangaroo is well coded, taking into account all WordPress coding standards as well as all of Google's AdSense rules & regulations.


* Use mobile optimized and responsive AdSense ads<br /> 
* Disable AdSense ads on phone, tablet or desktop devices<br /> 
* Define AdSense sizes for different devices<br />
* Use AdSense ads on posts, pages, categories or tags


= Plugin Options =

* Creates a WordPress widget in order to show Adsense ads
* Add ads on what pages you want.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the whole contents of the folder `ad_kangaroo` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress dashboard
3. Enjoy using it :)
